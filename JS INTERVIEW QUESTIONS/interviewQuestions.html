What is mutation? Which JavaScript data types are immutable?

Mutation refers to changing the state or value of an existing variable or object. In JavaScript, some data types are immutable, meaning that their values cannot be changed after they are created.
The following JavaScript data types are immutable:
Numbers
Booleans
Null
Undefined
Symbols
On the other hand, the following JavaScript data types are mutable:
Objects
Arrays
Functions
When you modify an immutable value, a new value is created rather than modifying the existing one. This is important to keep in mind when working with JavaScript, as it can help you avoid unexpected changes to your code.

Function arguments, also known as parameters, are values that are passed into a function when it is called. These values can be used inside the function to perform some task or computation.

What's the importance of return statement?

A return statement ends the execution of a function, and returns control to the calling function. Execution resumes in the calling function at the point immediately following the call. A return statement can return a value to the calling function.

What is a JavaScript event? Name few events.

In JavaScript, an event is a signal that is generated by the browser or the user

Click event - triggered when an element is clicked.
Keydown event - triggered when a key is pressed down.
Mouseover event - triggered when the mouse pointer moves over an element.
Scroll event - triggered when the user scrolls the page.
Submit event - triggered when a form is submitted.
Load event - triggered when the web page has finished loading.
Resize event - triggered when the window is resized.
