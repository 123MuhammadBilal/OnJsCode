some(): method will check if at least one element in the array meets the given condition. It returns a boolean value true if the condition is met for at least one element, otherwise false.
every(): method, on the other hand, checks if all elements in the array meet the given condition. It returns a boolean value true if the condition is met for all elements, otherwise false.
The forEach() method is used to execute a function on each element of an array, without creating a new array.
